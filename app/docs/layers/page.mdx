# Layers

> Layers are coming soon. This page describes the planned API.

Layers render data on the map — polygons, lines, points, tiles, and more.

## Planned Spec

```json
{
  "layers": {
    "buildings": {
      "type": "vector",
      "name": "Buildings",
      "data": "https://example.com/buildings.geojson",
      "style": {
        "fillColor": "#3498db",
        "opacity": 0.6,
        "lineColor": "#2980b9",
        "lineWidth": 1
      },
      "tooltip": ["name", "height"]
    }
  }
}
```

## Layer Types

- **`vector`** — GeoJSON features (points, lines, polygons) from inline data or URL
- **`mvt`** — Mapbox Vector Tiles from a tile URL template
- **`raster`** — XYZ raster tiles or images
- **`pmtiles`** — PMTiles archive from a `.pmtiles` URL
- **`hex`** — H3 hexagons via Deck.gl (future)

## Style Properties

- **`fillColor`** — fill color (static, continuous, or categorical)
- **`lineColor`** — stroke / outline color
- **`opacity`** — layer opacity 0–1
- **`lineWidth`** — stroke width in pixels
- **`pointRadius`** — point radius in pixels
- **`extruded`** — enable 3D extrusion
- **`elevationAttr`** — property name for extrusion height

## Data-Driven Styling

Colors can be static or data-driven. See the [Color System](/docs/colors) page for details on continuous and categorical color mapping.

```json
{
  "fillColor": {
    "type": "continuous",
    "attr": "temperature",
    "palette": "Sunset",
    "domain": [0, 40]
  }
}
```
