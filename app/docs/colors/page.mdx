# Color System

> The color system is coming soon. This page describes the planned API.

json-maps uses a flexible color system. Colors can be static values or data-driven mappings.

## Color Value Types

A `ColorValue` can be one of:

- **Static string** — `"#e74c3c"`, `"rgb(52, 152, 219)"`, `"steelblue"`
- **RGB array** — `[231, 76, 60]`
- **Continuous mapping** — numeric attribute to gradient
- **Categorical mapping** — string attribute to discrete colors

## Continuous

Map a numeric property to a color gradient using a palette.

```json
{
  "type": "continuous",
  "attr": "temperature",
  "palette": "Sunset",
  "domain": [0, 40],
  "steps": 7
}
```

- **`type`** — `"continuous"` (required)
- **`attr`** — feature property name (required)
- **`palette`** — CartoColor palette name (required)
- **`domain`** — `[min, max]` value range, auto-detected if omitted
- **`steps`** — color steps 2–12 (default: 7)
- **`nullColor`** — color for null/missing values

## Categorical

Map a string property to discrete colors.

```json
{
  "type": "categorical",
  "attr": "zone_type",
  "palette": "Bold",
  "categories": ["residential", "commercial", "industrial"]
}
```

- **`type`** — `"categorical"` (required)
- **`attr`** — feature property name (required)
- **`palette`** — CartoColor palette name
- **`categories`** — explicit category list
- **`nullColor`** — color for null/missing values

## Palettes

Uses [CartoColor](https://carto.com/carto-colors/) palettes (90+).

**Sequential**: Viridis, Sunset, SunsetDark, Mint, Emrld, Teal, BluYl, Peach, Burg, RedOr, Earth

**Categorical**: Bold, Pastel, Antique, Vivid, Prism, Safe

**Diverging**: TealRose, Geyser, Temps, Fall, ArmyRose, Tropic

## Examples

Choropleth with continuous color:

```json
{
  "layers": {
    "population": {
      "type": "vector",
      "data": "https://example.com/states.geojson",
      "style": {
        "fillColor": {
          "type": "continuous",
          "attr": "population",
          "palette": "Viridis",
          "domain": [500000, 40000000]
        },
        "opacity": 0.7,
        "lineColor": "#fff",
        "lineWidth": 0.5
      }
    }
  }
}
```

Zoning map with categorical color:

```json
{
  "layers": {
    "zones": {
      "type": "vector",
      "data": "https://example.com/zones.geojson",
      "style": {
        "fillColor": {
          "type": "categorical",
          "attr": "zone_type",
          "palette": "Bold"
        },
        "opacity": 0.6
      }
    }
  }
}
```
