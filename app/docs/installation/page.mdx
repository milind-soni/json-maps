# Installation

## npm

```bash
npm install json-maps maplibre-gl lucide-react
```

json-maps requires **React 18+** as a peer dependency.

## CSS

Import the MapLibre GL stylesheet in your app's entry point or layout:

```tsx
import "maplibre-gl/dist/maplibre-gl.css";
```

## Tailwind CSS

The component uses Tailwind classes internally for markers, legends, and popups. Add the package to your Tailwind content paths:

```js
// tailwind.config.js
content: [
  "./node_modules/json-maps/dist/**/*.js",
  // ...your other paths
]
```

## Usage

```tsx
import { MapRenderer } from "json-maps";
import "maplibre-gl/dist/maplibre-gl.css";

const spec = {
  basemap: "streets",
  center: [-73.98, 40.75],
  zoom: 12,
};

export default function MyMap() {
  return <MapRenderer spec={spec} />;
}
```

No API keys required. Basemaps are served free by [CARTO](https://carto.com/).

## Dependencies

- **[MapLibre GL](https://maplibre.org/)** — open-source map rendering engine
- **[Lucide React](https://lucide.dev/)** — icons for markers
- **[React](https://react.dev/)** — UI framework

## From Source

To run the playground and docs locally:

```bash
git clone https://github.com/milind-soni/json-maps.git
cd json-maps
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for the landing page, or [/playground](/playground) to edit specs live.
